About Big-O Project

Our application will be a tool for students to learn algorithm complexity analysis. Big-O notation is the format to express complexity analysis, this is where we get the Big-O Project name. Understanding algorithm complexity analysis and how to describe it using Big-O notation is a key component in computer science. The Big-O Project's goal is to give students a space and resources to learn complexity analysis and its role in programming efficiently. Furthermore, we want to provide teachers a way to integrate their classroom on online and be able to implement their curriculum onto the application. Overall, the Big-O Project wants to extend the classroom feel and the student-teacher relationship, while teaching complexity analysis.
