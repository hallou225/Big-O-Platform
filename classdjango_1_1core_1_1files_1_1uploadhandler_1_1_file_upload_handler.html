django.core.files.uploadhandler.FileUploadHandler.file_complete ( self,
file_size ) 
Signal that a file has completed. File size corresponds to the actual
size accumulated by all the chunks.

Subclasses should return a valid ``UploadedFile`` object.

Reimplemented in django.core.files.uploadhandler.TemporaryFileUploadHandler, and django.core.files.uploadhandler.MemoryFileUploadHandler.
